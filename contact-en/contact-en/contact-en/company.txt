		<div class="richtext">
            <div id="C_T_DefaultRegion_M7251_repEntries_ctl01_ctl00_etRichtext"><p><strong>IBB</strong><span> </span>is your construction partner for<span>&nbsp;</span><strong>building projects</strong><span>&nbsp;</span>from shell construction to turn-key completion of&nbsp;<span>residential</span><span></span>,<span> agricultural<a href="/agricultural-construction/" target="_blank">, </a></span><span>industrial</span>,<span>&nbsp;</span><span>commercial</span><span> </span>and<span> </span><a href="/industrial-commercial/public-sector/"><span>public </span>developments</a>. We guarantee security and quality for every<span> </span><a href="/projects/">construction project</a>.</p>
<h3>IBB Construction &ndash; since 1991</h3>
<p><strong>IBB Baugesellschaft<span>&nbsp;</span></strong>was founded in 1991. Today the<span>&nbsp;</span><strong>construction company</strong><span> </span>employs a staff of over 175 people in Belgium and Luxembourg.</p>
<p>At<span>&nbsp;</span><strong>IBB</strong><span>&nbsp;</span>we invest a lot of time and effort in the correct implementation of your ideas. Thanks to our in-house<span>&nbsp;</span><span>surveying office and technical department</span><span>, we are able to reschedule quickly in case of short-term complications</span>. As a result of careful project planning we are able to guarantee stable prices and timely completion for<span> all </span><strong>construction work</strong>.</p></div>

        //https://codyhouse.co/gem/vertical-fixed-navigation/

        $(document).ready(function ($) {

            var contentSections = $('.cd-section'),
                navigationItems = $('#cd-vertical-nav a');

            navigationItems.each(function (index) {
                $(this).attr("data-number", (index + 1));
                $(this).find('span').addClass("cd-label").before("<span class='cd-dot'></span>");
            });

            updateNavigation();
            $(window).on('scroll', function () {
                updateNavigation();
            });

            //smooth scroll to the section
            navigationItems.on('click', function (event) {
                event.preventDefault();
                smoothScroll(constructTarget($(this)));
            });

            function updateNavigation() {
                contentSections.each(function () {
                    $this = $(this);
                    var activeSection = $('#cd-vertical-nav a[href="#' + $this.attr('id') + '"]').data('number') - 1;
                    if (($this.offset().top - $(window).height() / 2 < $(window).scrollTop()) && ($this.offset().top + $this.height() - $(window).height() / 2 > $(window).scrollTop())) {
                        navigationItems.eq(activeSection).addClass('is-selected');
                    } else {
                        navigationItems.eq(activeSection).removeClass('is-selected');
                    }
                });
            }
            function constructTarget(targetElement) {
                var id = targetElement.attr('href').substring(1);
                return $("#" + $.escapeSelector(id))
            }
            function smoothScroll(target) {
                $('body,html').animate(
                    { 'scrollTop': target.offset().top },
                    600
                );
            }
        });

    </script>
